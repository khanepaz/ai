# تشخیص وضعیت کاغذ با هوش مصنوعی (PWA)

یک اپلیکیشن وب کاملاً آفلاین و قابل نصب روی گوشی که با چند تا عکس ساده که خودم از کاغذ گرفتم، یاد گرفته تشخیص بده کاغذ:

- تمیز و سالم  
- خط‌کشی شده  
- تا خورده  
- پاره یا آسیب‌دیده  

همه‌چیز با Teachable Machine گوگل ساخته شده و به صورت PWA روی گوشی نصب می‌شه.

### دمو زنده (مستقیم نصب کن)
https://khanepaz.github.io/ai

(توی گوشی باز کن → «افزودن به صفحه اصلی» یا «نصب» رو بزن)

### نحوه استفاده
1. روی لینک بالا کلیک کن  
2. یه عکس از کاغذ بگیر یا از گالری انتخاب کن  
3. چند ثانیه صبر کن → نتیجه رو با درصد احتمال می‌بینی

### ویژگی‌ها
- کاملاً آفلاین کار می‌کنه  
- روی اندروید و آیفون نصب می‌شه  
- بدون نیاز به اینترنت بعد از اولین بار  
- مدل با عکس‌های واقعی خودم آموزش دیده  

### فایل‌های پروژه
- `index.html` → صفحه اصلی  
- `my_model/` → مدل هوش مصنوعی (TensorFlow.js)  
- `manifest.json` + آیکون‌ها → برای تبدیل به PWA  

### می‌خوای خودت هم بسازی؟
1. برو به https://teachablemachine.withgoogle.com  
2. Image Project بساز  
3. از هر چیزی که دوست داری عکس بگیر (مثلاً برگ، پارچه، غذا، دستخط و ...)  
4. مدل رو به صورت TensorFlow.js خروجی بگیر  
5. کد من رو کپی کن و مدلت رو جایگزین کن  

هر وقت مدلت رو بهتر کردی، فقط پوشه `my_model` رو آپدیت کن → لینک همون لینک قبلی کار می‌کنه!
